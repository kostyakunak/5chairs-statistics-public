# 5 Chairs Statistics

A statistics dashboard application built with React and TypeScript.

## Features

- Real-time statistics visualization
- Interactive charts and graphs
- Modern, responsive UI

## Technologies

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Chart.js
- Supabase

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
